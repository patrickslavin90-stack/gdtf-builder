How to use GDTF Builder

  1. Upload an MA3 patch file (.xmlp) or type a channel list manually
  2. Select your fixture type (Moving Head / Static)
  3. Click Generate GDTF and download the file
  4. Copy the .gdtf file to your fixture library:
    USB: USB stick root > fixture types folder
    PC: C:\ProgramData\MALightingTechnology\gma3_library\fixturetypes
  5. In MA3, go to Patch > Fixture Types and refresh the library
  6. Your fixture should appear — patch and go!

Disclaimer: While this tool has been rigorously tested, mistakes can happen. Attributes can be reviewed and edited by uploading your output to GDTF-Share.com. Always verify critical fixtures before a live show.

What is a GDTF File?

GDTF (General Device Type Format) is the industry standard file format for describing DMX lighting fixtures. GDTF files contain DMX channel definitions, fixture geometry, gobo images, and colour wheel data. grandMA3 and MA3 consoles use GDTF profiles to patch and control intelligent lighting fixtures.

How to Create a GDTF Fixture Profile

Upload your DMX patch export, enter the fixture name and manufacturer, select the fixture type, and click Generate GDTF. The AI builds a fully compliant GDTF 1.2 profile with correct DMX channel functions, geometry references, and MA3-compatible attributes. Download as a .gdtf ZIP file ready to import.
